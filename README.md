# Employee Attrition Prediction using Machine Learning

## Project Overview
This project predicts whether an employee is likely to leave the company using machine learning techniques.

## Dataset
IBM HR Analytics Employee Attrition Dataset  
Source: Kaggle

## Model Used
Logistic Regression

## Evaluation Metrics
- Accuracy
- Confusion Matrix
- Classification Report

## How to Run
1. Open employee_attrition.ipynb
2. Run all cells sequentially
